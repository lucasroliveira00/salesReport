# Relatório de Análise de Vendas

Foi realizado um relatório de vendas utilizando como base de dados a Sample do Power BI.

## O que foi utilizado no relatório:

- Criação de layout
- Segmentação de Dados
- Botão com ação de troca de página
- Criação de indicadores
- Botão com indicador para alterar visualizações
- Seleção para agrupamentos e organização dos elementos
- Padronização de títulos
